# [GLOBEM website](https://the-globem.github.io/)

The official website for [GLOBEM](https://github.com/UW-EXP/GLOBEM).

The project uses
- [pnpm](https://pnpm.io)
- [Nextra](https://nextra.vercel.app).

```bash
pnpm install
pnpm dev
```

Visit `localhost:3000` to preview changes.
